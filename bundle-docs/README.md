# Bundle Docs

Kuratierte Referenzdokumente aus dem offiziellen XRechnung-Bundle.

## Struktur

- `bundle-docs/xrechnung/bundle/`
  Kuratierte Bundle-Uebersichten.
- `bundle-docs/xrechnung/model/`
  Kuratierte SeMoX-Modellreferenzen fuer fachliche Struktur und UBL-Bindings.
- `bundle-docs/xrechnung/testsuite/`
  Kuratierte Beispiele und Begleitdokumente aus der offiziellen Testsuite.
- `bundle-docs/xrechnung-3.0.2-bundle-*.zip`
  Lokale Ablage fuer das originale Upstream-Bundle zur bedarfsweisen Extraktion.
  Diese Archive sind nicht Teil des versionierten Repo-Inhalts.

## Einordnung

- Alles in diesem Bereich ist oeffentliche Referenz.
- Versioniert werden die kuratierten Referenzdateien fuer diese Arbeitsbasis.
- Das originale Bundle kann lokal in diesem Verzeichnis liegen, bleibt aber
  per `.gitignore` ausgeschlossen.
- Eigene Ableitungen, Templates und Hilfsartefakte liegen ausserhalb dieses
  Bereichs.
- Die Root-Lizenz dieses Repos gilt nicht automatisch fuer diese Fremdartefakte;
  massgeblich bleiben hier die jeweiligen Original-Lizenz- und
  Nutzungsbedingungen der Quelle.
